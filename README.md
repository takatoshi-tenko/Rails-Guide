- 他の構成要素
までを学習し終える

4月〜6月で

## プロジェクト方針
- 基本的にはmainブランチに直プッシュ
- 一つの作業単位でコミットを積む
- 気になるところには積極的にコメントを残す
  - コードの説明ではなく、それを書いた意図を記す
